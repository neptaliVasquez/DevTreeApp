Hey Neptali, you can simply pass the query param modules-page along with workspace-uuid. The value for modules-page must be:
is-sa if smart assist is to be enabled
default if smart assist is to be disabled
It is important to also pass default if smart assist is not enabled since the Workflows remembers the last UI change.
So, for example:
https://workflows-stage.grip-research.org/dashboard?workspace-uuid=%3CWORKSPACE_UUID%3E&modules-page=is-sa would launch the Workflows with the smart assist UI
https://workflows-stage.grip-research.org/dashboard?workspace-uuid=%3CWORKSPACE_UUID%3E&modules-page=default would launch the Workflows without the smart assist UI
